E-Commerce Customer Support Chatbot

This project is a Conversational AI chatbot designed to handle common customer support queries in an e-commerce platform. The chatbot classifies user queries into different business intents and provides appropriate responses.

Features

- Intent classification using Machine Learning
- Natural Language Processing (NLP)
- TF-IDF Vectorization
- LinearSVC classification model
- Rule-based intent detection for improved accuracy
- Interactive chatbot interface using Streamlit
- Handles multiple customer queries such as:
  - Order Management
  - Payment Issues
  - Account Management
  - Subscription Services
  - Service Requests

Machine Learning Pipeline

User Input  
↓  
Text Preprocessing  
↓  
TF-IDF Vectorization  
↓  
LinearSVC Model  
↓  
Intent Classification  
↓  
Response Generation  

Technologies Used

- Python
- Scikit-learn
- Natural Language Processing (NLP)
- TF-IDF Vectorizer
- LinearSVC
- Streamlit
